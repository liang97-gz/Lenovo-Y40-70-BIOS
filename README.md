# Lenovo Y40-70 BIOS 修改版合集

本仓库包含适用于联想 Y40-70 笔记本电脑的修改版 BIOS 固件文件（不含 EC）。所有文件均可用 CH341A 编程器刷入，请根据需求选择对应版本。

---

## 文件说明

| 文件名 | 对应版本 | 来源/说明 |
|--------|----------|-----------|
| **RMEXWinZIVY1X64.ROM** | 基于 1.11 版修改 | 在 `EXWinZIVY1X64.ROM` 基础上移除**网卡白名单**，可自由更换兼容网卡（如 WCBN808A） |
| **EXWinZIVY1X64.ROM** | 官方 1.11 版提取 | 提取自联想官方更新包 `99cn28ww`，未作修改 |
| **y40BIOS106** | 1.06 版（网友分享） | 早期版本，供特定需求用户使用 |

> ⚠️ **注意**：以上所有文件均**不包含 EC 固件**。如需更新 EC，请先通过官方 EXE 升级程序单独刷入 EC，再使用本仓库的 BIOS 文件进行刷写。

---

## 实测验证

- 本人使用 `RMEXWinZIVY1X64.ROM` 配合 CH341A 刷入，成功开机并识别 **WCBN808A** 网卡（移除白名单生效）。
- 三个BIOS文件均可用 CH341A 刷入。
